## Quick install

### NPM

```sh
npm install vue-fluent
```

### Yarn

```sh
yarn add vue-fluent
```

### Import

After installation, you can import the CSS/Sass file into your project using this snippet:

```sh
import 'vue-fluent/dist/vue-fluent.css'
import 'vue-fluent/dist/vue-fluent.js'
```
